# BUGJUMP-PixelSeven
一个类似Mojangles的字体，大写字母7个像素高，但支持更多字符! A font similar to Mojangles, with uppercase letters 7 pixels tall, but supporting more characters!

**Left**:向左偏移，适合用于Minecraft。 Shift to the left, suitable for use in Minecraft.

**Large**:更大的字形。 Larger font.

**LeftLarge**:向左偏移并有更大的字形，适合用于Minecraft搭配Modern UI。 Shifted to the left with a larger font, suitable for Minecraft with Modern UI.

该字体仍在不断制作中，之后还会补充更多! This font is still a work in progress, and more glyphs will be added later!
